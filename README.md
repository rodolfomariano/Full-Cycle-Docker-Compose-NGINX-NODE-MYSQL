# Full-Cycle-Docker-Compose-NGINX-NODE-MYSQL

## :rocket: Getting Started

First, clone the project:

```bash
git clone git@github.com:rodolfomariano/Full-Cycle-Docker-Compose-NGINX-NODE-MYSQL.git
```
Secund, enter the project folder

```bash
cd Full-Cycle-Docker-Compose-NGINX-NODE-MYSQL
```

Third, run the docker-compose.yaml

```bash
sudo docker-compose up -d
```
<h3>If all goes well, the project will be running on port 8080</h3>

<p>http://localhost:8080/</p>
